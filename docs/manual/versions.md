title: 版本比较
---

TQuant有两个版本，分别对应不同的用户需求。

- 个人本地版：

  支持本地实时行情、历史Tick数据，支持本地CTP交易接口，不支持多用户、多机使用，无需注册。
  如果需要使用使用TQuant提供的增值服务（互联网行情和交易服务），可以注册。
  
- 机构版：

  本地服务器部署方案，适合多人多机房使用，满足机构对合规性、安全性的要求。
