title: 版本比较
---

TQuant有三个版本，分别对应不同的用户需求。

- 个人版：仅使用本地行情和本地交易，手动维护本地历史数据。
- 个人VIP版：在个人版的基础上，使用TQuant提供的增值服务，包括互联网行情和交易服务、历史数据自动落地。
- 机构版：提供本地服务器方案，适合多人多机房使用，满足机构对合规性、安全性的要求。
